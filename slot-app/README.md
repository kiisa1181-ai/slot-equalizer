# 🎰 乗り打ち精算アプリ

スロットの乗り打ちでグループの収支を均等精算するPWAアプリです。

## iPhoneでアプリとして使う方法

### ① Vercelにデプロイ（無料・推奨）

1. [GitHub](https://github.com) でアカウント作成（未登録の場合）
2. 新しいリポジトリを作成して、このフォルダのファイルを全部アップロード
3. [Vercel](https://vercel.com) でGitHubアカウントでログイン
4. 「New Project」→ 作ったリポジトリを選択 → 「Deploy」
5. デプロイ完了後、発行されたURLをiPhoneのSafariで開く

### ② iPhoneのホーム画面に追加

1. SafariでアプリのURLを開く
2. 画面下部の **共有ボタン（四角から矢印が出てるアイコン）** をタップ
3. **「ホーム画面に追加」** をタップ
4. 名前を確認して「追加」→ ホーム画面にアイコンが追加される！

これでネイティブアプリと同じようにフルスクリーンで起動します。

---

```bash
npm install
npm run dev
```
